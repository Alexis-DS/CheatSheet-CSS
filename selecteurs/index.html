Quelques sélecteurs CSS les plus utiles

À quoi sert le CSS ?
background-color : Changer la couleur du fond du site.
margin-(top; left; right; bottom) : Mettre une marge entre différents objets.
padding-(top; left; right; bottom) : Faire un espacement d'un autre objet.
height : Hauteur.
width : Largeur.
border(-(top; left; right; bottom)) : Mettre une ou des bordure(s) à un tableau ou autres.
.class : Modifier quelque(s) chose(s) dans un ou plusieurs code(s).
#id : Équivaut à ".class" . Modifier quelque(s) chose(s) dans un ou plusieurs code(s).
font-size : Changer la taille du texte.
font-family : Changer la police du texte.
color : Changer la couleur du texte.
text-align : Aligner le texte à un endroit.
